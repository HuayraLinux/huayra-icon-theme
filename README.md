# Temas de Iconos de Huayra GNU/Linux

Este proyecto reune los sucesivos temas de íconos desarrollados para **Huayra GNU/Linux** por el equipo Huayra.

Incluye los siguientes temas:

* Huayra Violeta
* Huayra Mayo
* Huayra Limbo
* Huayra Verde Fresco
* Huayra Liso
* Huayra Ocho
